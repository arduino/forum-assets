Discussion about the **Arduino IDE 1.x** software

**Arduino IDE** is an integrated development environment for Arduino sketches. This application has gone through two major version series:

- **1.x**: Versions 1.0.0 - 1.8.19
- **2.x**: Versions >=2.0.0

Use this category for topics where the primary subject matter is any of the versions of **Arduino IDE** from the 1.x major version series (e.g., 1.8.19).

For discussions about **Arduino IDE** 2.x, please use the #development-tools:ide-2-x category.
