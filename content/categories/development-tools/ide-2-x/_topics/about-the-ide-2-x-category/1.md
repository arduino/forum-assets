Discussion about the **Arduino IDE** software

**Arduino IDE** is an integrated development environment for Arduino sketches.

Use this category for topics where the primary subject matter is any of the versions of **Arduino IDE** from the 2.x major version series (e.g., 2.3.7).

For discussions about versions of **Arduino IDE** from the 1.x series (e.g., 1.8.19), please use the #development-tools:ide-1-x category.
