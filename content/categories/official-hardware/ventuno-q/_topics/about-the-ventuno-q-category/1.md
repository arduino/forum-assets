Discussion about the **VENTUNO Q** board

The **Arduino VENTUNO Q** combines high-performance computing and real-time control.

Use this category for topics where the primary subject matter is the **Arduino VENTUNO Q** board.

The **VENTUNO Q** has a **Qualcomm Dragonwing IQ-8275** microprocessor running Linux, and an **STMicroelectronics STM32H5F5** microcontroller to run Arduino sketch programs.
